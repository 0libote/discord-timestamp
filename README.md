# Discord Timestamp Generator

A clean, efficient tool for generating dynamic Discord timestamps. Ensure your event times are accurate for everyone, regardless of their timezone.

## Features

- **Precise Date & Time Selection**: Intuitive controls to pick the exact moment.
- **Timezone Support**: Searchable database of all global timezones.
- **Instant Preview**: See exactly how the timestamp will look before you copy it.
- **One-Click Copy**: Quickly grab the code for any format (Relative, Short Time, Long Date, etc.).
- **History**: Automatically saves your recently generated timestamps for quick access.
- **Dark Mode**: Fully supported dark theme that matches Discord's aesthetic.

## Usage

1.  **Select Date & Time**: Choose the target date and time.
2.  **Set Timezone**: Specify the timezone for the selected time.
3.  **Copy Code**: Click the copy button next to your desired format (e.g., `<t:1728403200:R>`).
4.  **Paste**: Paste the code into your Discord message. It will automatically convert to the viewer's local time.

## Tech Stack

### Project Context

This application was developed as an experiment in AI-assisted software engineering. The majority of the codebase, including component logic, styling, and documentation, was generated by an AI coding assistant under human supervision. The goal was to evaluate the efficacy of AI in building functional, polished web applications with minimal human intervention.
